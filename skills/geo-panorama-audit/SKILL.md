# GEO 全景诊断（geo-panorama-audit）

## 目标

在5大主流AI平台上系统性地测试品牌的GEO表现，构建品牌GEO基准线，发现差距与机会，生成优先级行动计划。

## 测试平台

| 平台 | 开发方 | 特点 | 测试权重 |
|------|--------|------|---------|
| **豆包** | 字节跳动 | 抖音生态整合，C端覆盖广 | 25% |
| **文心一言** | 百度 | 中文搜索生态优势，百科/知道/贴吧整合 | 25% |
| **通义千问** | 阿里巴巴 | 电商/企业服务场景 | 20% |
| **DeepSeek** | 深度求索 | 开源模型，开发者/技术社区影响力大 | 15% |
| **Kimi** | 月之暗面 | 长文本处理能力强，B端/知识工作者覆盖 | 15% |

## 四大分析维度

### 1. 品牌召回率（Brand Recall Rate）

**目标**：当用户用核心关键词提问时，AI是否提及品牌？

**测试方法**：
- 准备核心关键词列表（品牌词 + 品类词 + 痛点词 + 场景词 + 地域词）
- 在5个平台上逐一测试
- 记录品牌是否被提及、提及位置（前1/3/5条）

**评分标准**：
| 召回等级 | 标准 |
|---------|------|
| 高 | 品牌在首条结果中出现且描述正面 |
| 中 | 品牌在后续结果中出现 |
| 低 | 品牌未被提及 |
| 偏差 | 品牌被提及但描述有误 |

### 2. 语义准确性（Semantic Accuracy）

**目标**：AI对品牌的描述是否准确、完整、正面？

**检查维度**：
- 品牌名称是否正确（全称/简称/拼写）
- 核心业务描述是否准确
- 差异化优势是否被正确呈现
- 是否存在负面关联或误导性描述
- 关键信息是否有遗漏

**输出**：逐平台语义准确性评分 + 错误描述汇总 + 缺失信息列表

### 3. 竞品定位（Competitor Positioning）

**目标**：在AI回答中，哪些竞品出现？品牌与竞品的相对位置如何？

**分析维度**：
- 在品类词提问中，竞品出现频率排序
- 品牌 vs 竞品的提及位置对比
- AI是否直接将品牌与特定竞品做对比？描述是否有利？
- 竞品的差异化优势是否被AI强调？

**输出**：竞品出现热力图 + 品牌-竞品相对定位矩阵

### 4. 内容源质量（Source Quality）

**目标**：AI引用哪些内容源来生成关于品牌/行业的回答？这些源的质量如何？

**分析维度**：
- 来源类型分布（百科/官网/自媒体/新闻/论坛/论文）
- 权威来源占比
- 负面/过时来源标记
- 品牌自有内容是否被引用？
- 引用来源的时效性

**输出**：来源类型分布图 + 权威来源率 + 内容源差距分析

## 工作流程

### Step 1：准备测试关键词矩阵

根据品牌资料包，生成覆盖5个维度的测试关键词矩阵：
- **品牌词**（5-10个）：品牌名、子品牌、创始人、核心产品名
- **品类词**（5-10个）：行业通用品类描述
- **痛点词**（5-10个）：目标用户的典型问题/需求表述
- **场景词**（5-10个）：使用场景/应用场景
- **地域词**（3-5个）：城市+品类、区域+服务

### Step 2：逐平台测试

对每个平台执行以下测试：
1. 逐一输入测试关键词
2. 记录完整回答内容
3. 提取品牌提及情况
4. 提取竞品出现情况
5. 记录引用的内容源

需要联网搜索能力以获取跨平台测试结果。

### Step 3：数据汇总与分析

- 汇总5平台 × 每平台N个关键词的测试结果
- 计算品牌召回率、语义准确率
- 绘制竞品出现频次排序
- 分析内容源质量分布

### Step 4：生成全景诊断报告

## 输出物

### 1. GEO基准线报告

以结构化报告形式呈现：
- **执行摘要**：一句话概述品牌GEO整体表现
- **各平台表现评分卡**：四大维度雷达图
- **详细分析**：逐平台（豆包/文心一言/通义千问/DeepSeek/Kimi）× 四维度（召回率/语义准确性/竞品定位/内容源质量）
- **核心发现**：TOP3发现 + TOP3问题

### 2. 差距分析报告

- 当前状态 vs 目标状态的量化差距
- 差距根因分析（内容缺失/权威性不足/平台覆盖不足/竞品强势）
- 差距影响评估（对品牌认知/用户获取/转化的影响）

### 3. 优先级机会地图

- 按"影响程度 × 修复难度"矩阵分类的机会点
- 速效机会（高影响×低难度）→ 优先执行
- 战略投资（高影响×高难度）→ 列入计划
- 常规维护（低影响×低难度）→ 持续优化
- 待观察（低影响×高难度）→ 暂缓

### 4. 竞品态势图

- 各平台品牌 vs TOP5竞品的相对位置
- 竞品优势维度分析
- 品牌可抢占的差异空间

## 使用示例

```
用户：帮我做「某口腔连锁品牌」的GEO全景诊断
→ 前提：已完成geo-brand-intake，有品牌资料包
→ 生成关键词矩阵（品牌词/品类词/痛点词/场景词/地域词）
→ 在5大平台逐一测试
→ 分析召回率/语义准确性/竞品定位/内容源质量
→ 输出全景诊断报告
```

## 前置依赖

- 必须在 `geo-brand-intake` 之后执行（需要品牌资料包中的15个核心字段）
- 需要联网搜索能力
- 需要直接访问5大AI平台进行测试

## 与GEO技能链的关系

`geo-panorama-audit` 是GEO端到端工作流的**第二步**：
- ← `geo-brand-intake`：提供品牌事实基础
- → `geo-execution-roadmap`：诊断结果转化为执行路线图
- → `geo-question-discovery`：基于诊断中的关键词表现挖掘优化方向
- → `geo-article-factory`：针对诊断发现的差距进行内容补充
