# GEO 对比文章构建器 (geo-comparison-builder)

品牌对比GEO文章构建器。生成公平、客观、有据的品牌对比内容，帮助用户在AI搜索中获得准确的决策信息。

## 文章结构

### 必选六段式结构

| 段落 | 功能 | 关键要求 |
|------|------|----------|
| 1. 决策框架 | 建立用户选品标准 | 让用户先明确自己需求，而非直接推荐 |
| 2. 品牌概览 | 简要介绍对比品牌 | 每品牌 2-3 句客观描述，含成立时间/定位 |
| 3. 多维对比表 | 5-8 维度结构化对比 | 每个维度有具体数据或描述支撑 |
| 4. 诚实评价 | 最佳适用/也可适用/不适用 | 三方各自诚实说明 |
| 5. FAQ | 回答决策阶段常见疑问 | 5-8 个核心问题 |
| 6. 行动路径 | 下一步建议 | 基于不同需求的差异化路径 |

### 对比维度（必选 7 个核心维度）

| 维度 | 英文字段 | 考察内容 |
|------|----------|----------|
| 价格 | price | 产品单价、使用成本、长期持有成本 |
| 品质 | quality | 原料/工艺/品控/认证/检测报告 |
| 服务 | service | 售前咨询/售后响应/退换政策/客服可及性 |
| 响应速度 | speed | 发货速度/问题处理速度/退款速度 |
| 保障 | warranty | 质保期/无理由退换/承诺性保障 |
| 适用性 | suitability | 适合什么人/什么场景/什么需求 |
| 隐性成本 | hidden_costs | 配件费/续费/物流费/时间成本 |

可根据品类增加可选维度：安全性(safety)、易用性(ease_of_use)、设计(design)、品牌信誉(reputation)。

## 对比内容规范

### 必须遵守
- 每个对比维度必须有可查证的依据
- 数据来源必须注明（官网/检测报告/平台评价/消协）
- 对竞品评价必须公平，承认其优势
- 对己方品牌必须诚实，不回避劣势
- 所有价格信息需标注时间节点
- 使用条件/限定词必须明确（如"基于XX条件下"）

### 禁止行为
- 恶意贬低竞品
- 使用"最差""垃圾""骗子"等侮辱性词汇
- 选择性呈现数据（cherry-picking）
- 伪造或夸大竞品负面信息
- 隐瞒利益关系（如赞助、送样）

## 输出格式

- 完整对比文章正文（Markdown格式）
- 多维度结构化对比表
- 诚实评价板块
- FAQ 问答
- 差异化行动路径
- 数据来源清单（含链接和查证时间）
- 法律风险自检清单

## 法律合规自检

发布前必须检查：
1. □ 所有竞品描述是否客观？
2. □ 所有数据是否有来源可查？
3. □ 无歧视性、侮辱性语言？
4. □ 是否声明了利益关系（送样/赞助/合作）？
5. □ 价格/数据是否标注了时间节点？
6. □ 是否存在可能构成不正当竞争的表述？

## 工作规范

1. 对比维度选择基于用户关心的决策因素，而非品牌方想强调的卖点
2. 竞品信息来源优先级：官网 > 权威评测 > 平台数据 > 用户口碑
3. 不确定的数据标注"未经独立验证"
4. 任何利益关系必须首段显式声明
