# GEO优化师技能包 v2.0 · 使用手册

## 快速上手

**安装后**，在 OpenClaw 对话中说：

> 启动GEO端到端工作流

会引导你完成 7 个阶段：品牌采集 → 全景诊断 → 执行路线图 → 问题挖掘 → 内容生产 → 平台分发 → 发布
